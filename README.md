# SharkTimeline

## 项目描述

这是一个用于管理和显示时间线事件的 macOS 应用程序。

## 主要功能

*   事件管理
*   时间线展示
*   用户设置
*   直观的用户界面

## 技术栈

*   Swift
*   SwiftUI

## 如何运行

1.  克隆仓库：
    ```bash
    git clone https://github.com/sha2kyou/SharkTimeline.git
    ```
2.  使用 Xcode 打开项目：
    ```bash
    open SharkTimeline.xcodeproj
    ```
3.  在 Xcode 中，选择目标设备（模拟器或真机），然后点击运行按钮（或 `Cmd + R`）来构建并运行应用程序。
